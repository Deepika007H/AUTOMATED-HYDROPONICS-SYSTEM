\# Automated Hydroponics System (ESP8266)



An IoT-based hydroponics automation system using ESP8266. It monitors pH, TDS, temperature, humidity, water level and controls pumps and lighting. Data logs to Google Sheets via Apps Script, and a local dashboard displays real-time values.

\## Setup Instructions



1\. Open the firmware in Arduino IDE.

2\. Create `secrets.h` file in `/firmware/` (see `secrets.example.h`).

3\. Upload to ESP8266.



\## Sensors \& Actuators

\- pH probe

\- TDS sensor

\- DHT sensor (temp/humidity)

\- Float level sensor

\- Relay Controlled Pumps

\- Grow Light Control
